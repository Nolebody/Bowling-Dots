
# Bowling Dots PWA v8b
- Live counters with sticky first column
- Archive with per-night snapshots
- Totals tab (by bowler / by category / by night)
- Settings tab to edit categories & rates
- Offline-ready PWA

## Deploy (GitHub Pages)
1. Upload index.html, sw.js, manifest.webmanifest and the icons/ folder to your repo root.
2. Commit to main. Open https://<yourname>.github.io/Bowling-Dots/?v=8b
